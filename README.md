# Deep Learning Intrusion Detection System

## 📌 Project Overview
This project is a Deep Learning-based Intrusion Detection System that detects malicious network traffic using a Convolutional Neural Network (CNN).

## 🚀 Features
- Detects network intrusions
- Uses CNN for classification
- Python-based implementation
- Generates prediction results

## 🛠️ Technologies Used
- Python
- TensorFlow
- Keras
- NumPy
- Pandas
- Matplotlib
- Scikit-learn

## 📂 Project Structure

Dataset/
model/
DeepLearningIntrusion.py
requirements.txt
test.py
output.html
run.bat

## ▶️ How to Run

1. Install dependencies

```bash
pip install -r requirements.txt
```

2. Run the project

```bash
python DeepLearningIntrusion.py
```
